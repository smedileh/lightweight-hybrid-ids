# A Lightweight Hybrid Intrusion Detection System (IDS) for Edge Network Security

## Abstract:
>The proliferation of Internet of Things (IoT) devices and the expanding attack surface of small-scale networks have created a pressing demand for lightweight, cost-effective cybersecurity solutions. This paper presents a \emph{hybrid Intrusion Detection System} (IDS) that combines signature-based detection (Snort, Suricata) with a machine-learning pipeline—comprising Decision Tree, Random Forest, K-Nearest Neighbours (KNN), and Long Short-Term Memory (LSTM) classifiers—deployed on a Raspberry Pi~5 edge device. The system was trained on the CSE-CIC-IDS2018 benchmark dataset (3.4~million labelled flow records) and evaluated against seven categories of simulated attacks, including Denial-of-Service (DoS) variants, port scanning, brute force, and the Heartbleed TLS exploit. The trained Random Forest model achieved an overall accuracy of 98.05\%, while the LSTM reached 98.08\%. In live network tests, the hybrid architecture attained a detection rate of 97\% with a false-positive rate of only 5\%, outperforming Snort (92\%) and Suricata (95\%) individually—particularly for slow, stealthy application-layer attacks. The modular architecture is validated as feasible, energy-efficient, and extensible for real-world IoT and small-office/home-office (SOHO) environments.
